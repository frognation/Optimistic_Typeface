## A - Journy
### A. 컨텍스츄얼


프롬프트: 

타입디자인할때 컨텍스츄얼 타입 기능 이용해서
그냥 t하나 나왔을때는  t 가 나오고
tt를 타입했을때는 "t + t.c2  로 표시되고,
ttt를 타입했을때는 "t + t.c1 + t.c2" 로 표시되고,
4개이상 타입했을때는 "t + t.c1 를 n-2개넣고 + tc2"로 표시되게 하려면 어떻게할까?
그리고, 여기서 t.c1, t.c2는 모두 임시이름임. 어떻게 이름짓는게 표준에맞고 관리하거나 스크립트짤때좋은지 알려줘
그리고 이걸 여러글자에 적용할거야.
